Smartshare - Rust, Typescript, Lua 2024

Smartshare is a collaborative editing systemn designed from to start to be cross-editor. The project provides a server, a client and adapter plugins for VSCode and NeoVim. The goal of this project is to make adding support for a new editor as esay as possible easy by requirering to adapt only a basic communication protocol between the text editor and a local running process, this mechanism is inspired by the Language Server Protocol design.
